# Apps Script — Script Properties

Configure these values in:

Apps Script → Project Settings → Script Properties

## Production
```text
ENVIRONMENT=production
SPREADSHEET_ID=YOUR_PROD_SPREADSHEET_ID
STORAGE_FOLDER_ID=YOUR_PROD_STORAGE_FOLDER_ID
EMAIL_TO=YOUR_EMAIL
TELEGRAM_CHAT_ID=YOUR_TELEGRAM_CHAT_ID
TELEGRAM_TOKEN=YOUR_PROD_BOT_TOKEN
```
## Development
```text
ENVIRONMENT=development
SPREADSHEET_ID=YOUR_DEV_SPREADSHEET_ID
STORAGE_FOLDER_ID=YOUR_DEV_STORAGE_FOLDER_ID
EMAIL_TO=YOUR_EMAIL
TELEGRAM_CHAT_ID=YOUR_TELEGRAM_CHAT_ID
TELEGRAM_TOKEN=YOUR_DEV_BOT_TOKEN
```
